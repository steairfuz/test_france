Test 1 codage module Python 3 :

Développer un module qui permettra à l’utilisateur de saisir ça ville sur un premier écran. Pour cela on utilisera l’auto-complétion afin d’éviter tout type de saisir erroné, puis pour améliorer la performance on se connectera à un API open source des villes de France. Sur la deuxième page on affiche les données suivantes : la ville , le tarifs, la longitude et la latitude de la ville ainsi que le nombre d’habitants ;

  • Habitant inférieur à 1000 le prix est de 5€
  • Habitant entre 1000 à 9000 le prix est de 10€
  • Habitant entre 9000 à 30 000 le prix est de 30€
  • Habitant supérieur à 30 000 le prix est de 50€
